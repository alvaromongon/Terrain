This solution implements the terrain generator using type script.

- Contains:
	- BabylonJS: (https://github.com/BabylonJS/Babylon.js/tree/master/Tools/BuildOurOwnBabylonJS/BabylonJS)
		1.- Download the las version of babylon.js repository (https://github.com/BabylonJS/Babylon.js)
		2.- Add the reference into the main solution (TerrainTS) to the project
		3.- Follow this steps to reference the ts files: http://frankcode.wordpress.com/2014/02/02/cross-project-typescript-unit-testing-part-1/
	- TerrainGenerator.Web: contain the web site with the typescript code and reference to the back-end
